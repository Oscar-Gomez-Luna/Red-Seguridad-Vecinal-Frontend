# Red Seguridad Vecinal - Frontend (React + Vite)

Aplicación Web Progresiva (PWA) diseñada para la administración integral del fraccionamiento.

## Documentación del Proyecto
Puedes consultar el documento técnico completo aquí:
 **[Documento Final - Red de Seguridad Vecinal.pdf](https://github.com/Oscar-Gomez-Luna/Red-Seguridad-Vecinal-Frontend/blob/master/Documento%20Final%20-%20Red%20de%20Seguridad%20Vecinal.pdf)**

## Stack Tecnológico
* **Core:** React + Vite
* **Estilos:** Tailwind CSS
* **Mapas:** Google Maps API interactivo

## Módulos Principales
* **Dashboard:** Métricas de incidentes y finanzas en tiempo real.
* **Gestión de Usuarios:** Altas, bajas y perfiles de residentes y seguridad.
* **Mapa de Incidentes:** Visualización geoespacial de reportes.
* **Tesorería:** Generación de cargos y control de pagos.
